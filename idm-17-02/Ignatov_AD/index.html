Тег Описание тега
1) <a> a) Тег-контейнер для разделов HTML-документа. Используется для группировки блочных элементов с целью форматирования стилями.
2) <b>b) Создаёт гипертекстовые ссылки.
3) <body>c) Заголовок HTML-документа, отображаемый в верхней части строки заголовка браузера.
4) <br>d) Задает полужирное начертание отрывка текста, не придавая акцент или важность выделенному.
5) <button>e) Перенос текста на новую строку.
6) <div>f) Создает интерактивную кнопку. Внутрь тега можно поместить содержимое — текст или изображение.
7) <head>g) Элемент маркированного или нумерованного списка.
8) <img>h) Элемент-контейнер для метаданных HTML-документа, таких как<title>, <meta>, <script>, <link>, <style>.
9) <p>i) Тег для создания таблицы.
10) <table>j) Встраивает изображения в HTML-документ с помощью атрибута src, значением которого является адрес встраиваемого изображения.
11) <title>k) Параграфы в тексте.
12) <li>l) Упорядоченный нумерованный список. Нумерация может быть числовая или алфавитная
- m) Представляет тело документа
- n) Расставляет акценты в тексте, выделяя полужирным.
