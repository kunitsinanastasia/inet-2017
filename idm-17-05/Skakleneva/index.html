 Лабораторная работа №2 "Разработка простого веб-приложения"

Требования:
Проектирование и разработка индивидуального или коллективного веб-приложения с использованием html,css,js + json,xml
Выполнено:
wtf?! Веб-приложение "Strike game 2.0" представляет собой эмуляцию простейшего однопользовательского шутера. Для каждой новой сессии игра хранит счетчик уже сыгранных партий.

Приложение реализовано на js с использованием шаблона проектирования MVC, т.к. он хорошо подходит для реализации клиентских приложений.
Контроллер в данной модели отвечает за реакцию на действия пользователя. Функция контроллера init() запускает приложение и реагирует на каждое действие игрока - клик или движение мышкой. В зависимости от запроса со стороны пользователя контроллер обращается к одной из трех функций модели startGame(), playerMove() или update(). Модель в данном приложении отвечает за обновление координат всех объектов на поле, обсчет коллизий и ведение счета игры. Иначе говоря, модель управляет всеми данными приложения. Вид же отвечает на отрисовку всех объектов на поле с помощью функции draw(), запуск которой инициирует контроллер.

: HTTP/2 (изначально HTTP/2.0) — вторая крупная версия сетевого протокола HTTP, используемая для доступа к World Wide Web. Основное внимание в протоколе уделяется производительности; в частности, задержка ожидания конечного пользователя, использование ресурсов сети и сервера. В мае 2015 года спецификация HTTP/2 была опубликована как RFC 7540. По данным W3Techs на 1 декабря 2017 года, 22,3 % всех веб-сайтов поддерживают протокол HTTP/2.
