Данная лабораторная работа представляет собой настройку небольших сетей.
Имеется три сети, в каджой из которых по одному коммутатору и два компьютера.
Каждой сети соответствует свой маршрутизатор.
Для всех сетей задан свой пул адресов с определенными масками сети в соответствии с вариантом.
Для всех устройств были установлены IP-адреса из заданного пула.
Была настрона статическая маршрутизация к одной из сетей.
При добавлении в рабочее пространство двух коммутаторов с их компьютерами
и соединении этих коммутаторов кабелем, пакеты отправленные из одной сети
не могли попасть в другую, поскольку для каждого из коммутаторов были настроены свои адреса и маски.
для обеспечения взяимодействия двух сетей используют маршрутизаторы.

Подготовка к экзамену Самооценка прохождения теста по инновациям, формулировка одного вопроса по теме "Интернет-технологии" в одной из форм, встречающихся в тесте. Презентации лекций и тесты расположены на Яндекс-Диске.
